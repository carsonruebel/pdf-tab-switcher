Easily embed and toggle between two PDF files in your Elementor layout. Built for resumes, portfolios, menus, reports, and more.

== Description ==

**PDF Tab Switcher** is a lightweight Elementor widget that extends the capabilities of the popular [PDF Embedder](https://wordpress.org/plugins/pdf-embedder/) plugin.

It allows you to display **two PDF files with tab switching**, perfect for showcasing:

- Resumes with a “CV” vs “Resume” layout
- Multi-language menus or brochures
- Reports or reference material in dual formats
- Before/After comparisons or alternate versions of the same document

The widget uses a compact, responsive design with tab buttons and smooth fade transitions.

**Requirements:**  
This plugin depends on the free **PDF Embedder** plugin for rendering PDF files.

== Installation ==

1. Upload the plugin ZIP via Plugins → Add New → Upload Plugin.
2. Activate the plugin.
3. Edit a page with Elementor and drag the “PDF Switcher" widget into any section.
4. Upload two PDFs and customize the tab labels.

== Frequently Asked Questions ==

= Can I change the tab names? =
Yes, the widget settings let you customize the label text for each tab.

= Do I need any other plugins? =
Yes, the free **PDF Embedder** plugin is required for the PDFs to display properly.
